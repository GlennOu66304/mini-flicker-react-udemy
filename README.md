# Udemy-From-Zero-To-Build-a-Project
1.Type npm install -g create-react-app to Build React Project Enviroment 
```
If you use npm 5.1 or earlier, you can't use npx. Instead, install create-react-app globally:

npm install -g create-react-app

Now you can run:

create-react-app mini-flicker 
```
2. Run the Code in Browser:
```
cd mini-flicker 
npm start
```
